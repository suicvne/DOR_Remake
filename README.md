## DOR Remake Bug Reports

Please submit bug reports for the game here.

Be sure to include crucial information such as:

- Client Version
- Action Performed to cause the bug
- Operating system
- Specs
- Reproducability?
- Screenshots

This will aid the team in fixing bugs as they come in.
